About Epistery Scan

Epistery Scan indexes the signed web — websites that publish authored, cryptographically signed data via the AI Discovery Standard.

Unlike traditional search engines that scrape and repackage content, Epistery only indexes what organizations have explicitly published and signed. Blockchain provides the trust substrate: signature verification, identity binding, and provenance.

How to use Search

Type a keyword, organization name, or domain into the search bar on the home page. Results come from two sources:

  • Signed Web — domains that publish a /.well-known/ai manifest, indexed and verified by Epistery
  • MCP Registry — thousands of Model Context Protocol services, searchable by name, category, or capability

Results are labeled with trust badges: Verified (signature matches on-chain identity), Signed (cryptographically signed but not yet chain-verified), or Open (published but unsigned).

Delegate to a live service

Prefix your query with @service-name to route it directly to a live MCP service. Epistery picks the best-matching tool on that service and calls it on your behalf. Browse available services on the MCP Tools page.

Submit your domain

Publish a /.well-known/ai manifest on your domain and enter the domain name in search. Epistery will discover, index, and display it. You can also use the Discovery page to check any domain.

Sample queries

blockchain — find organizations publishing about blockchain technology identity — discover identity and authentication services rootz.global — look up a specific domain's manifest and trust status @stripe — delegate a query to the Stripe MCP service @github — delegate a query to the GitHub MCP service AI Discovery — search indexed concepts across all signed domains
