Developers

Epistery Scan exposes a JSON API for AI agents and developers. Every page also supports content negotiation — send Accept: application/json to get JSON from any URL.

AI Discovery

GET /.well-known/ai — machine-readable manifest

Search

GET /api/search?q={query} — knowledge search
GET /api/search/stats — index statistics
POST /api/search/submit — submit a domain

Discovery

GET /api/discovery — list indexed domains
POST /api/discovery — check a domain

Feed

GET /api/feed — recent changes

MCP Registry

GET /api/mcp/categories — service categories
GET /api/mcp/search?q={query} — search MCP services
GET /api/mcp/service/{name} — service detail

Full details in the AI Discovery manifest.
